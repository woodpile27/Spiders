# QQmusic
周董在唱什么

## 主要的python库

- python3.5
- selenium
- jieba
- wordcloud

## 各模块功能

### spider.py
通过selenium和requests在qq音乐上爬取周杰伦每张专辑的所有歌曲并保存在文件中。

### ciyun.py
读取歌词并通过jieba分词和wordcloud生成词云。
